# 🍺 飲み会リマインダー

飲み会中に彼女からのLINEを確認するためのリマインドアプリ（PWA）。

## 機能

- 「今から飲みに行く」ボタンで即スタート
- 30 / 45 / 60 分のリマインド間隔を選択可能
- 時間になるとスマホに「LINEチェック」通知を表示
- 経過時間・次回リマインドまでのカウントダウン・通知履歴を表示
- ホーム画面に追加してネイティブアプリのように使える（PWA）

## 使い方

1. ブラウザで `index.html` を開く（またはGitHub Pagesで公開）
2. 通知の許可を求められたら「許可」をタップ
3. リマインド間隔を選んで「今から飲みに行く」をタップ
4. タイマーが動き始め、設定した間隔でLINEチェック通知が届く
5. 飲み会終了後は「飲み会終了」ボタンをタップ

## GitHub Pages での公開

```
Settings → Pages → Source: main branch / root
```

公開後は `https://<username>.github.io/<repo>/` でアクセス可能。

## ファイル構成

```
index.html   — メインアプリ
manifest.json — PWA設定
sw.js        — Service Worker（オフライン対応・通知）
```

## 注意

- 通知はブラウザの許可が必要です
- iOSでは「ホーム画面に追加」してから使うとバックグラウンド通知が安定します
- ブラウザのタブを閉じると通知が届かない場合があります（ホーム画面追加を推奨）
